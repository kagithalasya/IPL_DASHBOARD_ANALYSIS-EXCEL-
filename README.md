# 🏏 IPL Data Analysis Dashboard (2008–2025)

## 📌 Project Overview
This project is an interactive Excel Dashboard built using IPL data from 2008–2025. The dashboard provides insights into tournament winners, runners-up, toss decisions, venue statistics, trophies won by teams, and Man of the Match performances.

## 🎯 Objectives
- Analyze IPL season-wise performance.
- Compare toss decisions and match outcomes.
- Identify top-performing venues.
- Track championship wins across teams.
- Visualize Man of the Match achievements.

## 📊 Dashboard Features
- ✅ Season-wise filtering using slicers
- ✅ Winner and Runner-up tracking
- ✅ Man of the Season details
- ✅ Toss decision analysis (Bat First vs Field First)
- ✅ Top venues performance analysis
- ✅ Team trophy distribution
- ✅ Man of the Match statistics

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning & Transformation
- Dashboard Design
